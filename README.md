# 구매 협력사 성과·리스크 분석 (2주차)

구매(조달) 5종 합성(synthetic) CSV 데이터를 pandas로 조인·집계하고, Streamlit 단일 페이지
아코디언 대시보드로 분석 결과와 개선 제언을 제공합니다. 항목별 "열기"를 누르면 별도 페이지로
이동하지 않고 카드 바로 아래에서 내용이 펼쳐지며, "닫기"를 누르면 접힙니다.

- **데이터**: 입찰공고 260건 · 비딩참여 1,193건 · 프로젝트수행 260건 · 협력사 120개 · 협력사평가이력 720건
- **분석(1~7)**: 저가입찰, 신용등급, 기업규모, 업종별 리스크 / 평가지표 추이 / 신용도-해지율 상관 / 고위험 워치리스트(스코어보드)
- **제언(8~10)**: 협력사 리스크 스코어링 모델 / 저가입찰 제도 개선안 / 실행 로드맵

## 구조

```
app.py              진입점 - KPI 헤더 + 분석 항목 10개를 아코디언 카드로 렌더링
pipeline.py          pandas 데이터 파이프라인 (raw_data/*.csv → 조인·집계·리스크 스코어링)
data.py              pipeline 결과를 앱 전역 상수로 노출 (ANALYSIS_ITEMS 메타데이터 포함)
common.py            공통 스타일, 차트, 표 컴포넌트
sections.py          분석 항목 1~10번 각각의 본문(차트+해석 캡션+제언) 렌더링 함수
raw_data/            원본 CSV 5종
```

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 데이터 안내

`raw_data/`의 CSV는 합성(synthetic) 데이터이며 실제 기업·거래 정보가 아닙니다.
